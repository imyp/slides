The gradient

Given a function \(f:\mathbb{R}^n\rightarrow\mathbb{R}\),

we have a gradient operator \(\nabla_{\boldsymbol{x}}\)

defined such that \(\nabla_{\boldsymbol{x}} f(\boldsymbol{x})_i = \frac{\partial}{\partial x_i}f(\boldsymbol{x})\).

Example

Given a function \(f:\mathbb{R}^3\rightarrow\mathbb{R}\)

defined as \(f(\boldsymbol{x})=\frac{3}{x_1}-\exp(x_2) + {x_3}^2\)

what is the value of \(\nabla f(\boldsymbol{x})_2\)?

\[\boxed{\nabla f(\boldsymbol{x})_2 = \tfrac{\partial}{\partial x_2} (\tfrac{3}{x_1}-\exp (x_2) + {x_3}^2 ) = -\exp(x_2)}\]

Similarly it can be shown that \(\nabla f(\boldsymbol{x})= \left( \begin{smallmatrix} -3{x_1}^{-2} \\ -\exp (x_2) \\ 2x_3 \end{smallmatrix}\right)\)

The Jacobian matrix

Given a function \(\boldsymbol{f}:\mathbb{R}^m \rightarrow \mathbb{R}^n\)

we have a Jacobian matrix \(\boldsymbol{J} \in \mathbb{R}^{n\times m}\) of \(\boldsymbol{f}\)

defined such that \(J_{i,j}=\frac{\partial}{\partial x_j } f( \boldsymbol{x} )_i\).

Example

Given a function \(\boldsymbol{f}:\mathbb{R}^2\rightarrow \mathbb{R}^2\)

defined as \(\boldsymbol{f}(\boldsymbol{x})=\begin{pmatrix} 2x_1+x_2\\ x_1 - {x_2}^2 \end{pmatrix}\)

what is the value of \(J_{1,2}\)?

\[\boxed{J_{1,2}=\tfrac{\partial}{\partial x_2} (2x_1+x_2) = 1}\]

Similarly it can be shown that \(\boldsymbol{J}=\begin{pmatrix}2&1\\ 1&-2x_2 \end{pmatrix}\)
